## Code Challenge: First Unique Character in a String
Given a string, find the first **non-repeating** character in it and return its **index**. If it doesn't exist, return -1.

#### Example 1:
```
Input: "leetcode"
Output: 0
```

#### Example 2:
```
Input: "loveleetcode"
Output: 2
```

Source: Leetcode

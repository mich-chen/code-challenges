## Code Challenge: Palindrome Number

You are given an integer `x`, return `true` if `x` is palindrome integer. 

**Follow up:** Could you solve it without converting the integer to a string?

#### Example 1:
```
Input: x = 1881
Output: True
```

#### Example 2:
```
Input: x = -909
Output: False
```

#### Example 3:
```
Input: x = 10
Output: False
```

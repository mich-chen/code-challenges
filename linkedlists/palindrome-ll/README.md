## Code Challenge: Palindrome Linked List
Given a singly linked list, determine if it is a palindrome.

#### Example 1:
```
Input: [1->2]
Output: False
```

#### Example 2:
```
Input: []
Output: True
```

#### Example 3:
```
Input: [1->2->2->1]
Output: True
```
